# Buselligence Platform

## Category

**Open AI Empowerment Platform** — The Open Source AI Operating System

## Architecture

```
                    Buselligence
                         │
                      AI Core
         ┌───────────────┼───────────────┐
         │               │               │
    Learn            Create          Analyze          Build
         │               │               │
    Code              Data           Automate         Agents
```

## Platform layers

| Layer | Modules |
|-------|---------|
| **AI Core** | Universal assistant, model router, BYOK providers |
| **Workspace** | Conversations, documents, projects, knowledge base |
| **Create** | App builder, visual studios, document generation |
| **Analyze** | Data intelligence, semantic layer, connectors |
| **Build** | Developer studio, software engineer, deployment |
| **Automate** | Workflow builder, scheduled intelligence, MCP |
| **Agents** | 17+ specialized agents, marketplace, community |
| **Learn** | Adaptive learning system, study tutor agent |
| **Connect** | MCP-native architecture, data connectors |

## API

| Endpoint | Description |
|----------|-------------|
| `GET /api/platform/manifesto` | Mission, philosophy, manifesto text |
| `GET /api/platform/pillars` | Learn/Create/Analyze/Build pillars |
| `GET /api/workspace` | User workspace overview |
| `GET/POST /api/knowledge` | Knowledge engine |
| `GET/POST /api/learning/sessions` | Learning system |
| `GET /api/agents` | All AI agents (personal, business, dev) |
