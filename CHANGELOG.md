# Changelog

All notable changes to Buselligence are documented here.

Format based on [Keep a Changelog](https://keepachangelog.com/).

## [8.0.0] - 2026-07-16

### Added

- **Buselligence Kernel** — unified execution layer (identity, context, permissions, memory, tools, agents, models, events, execution)
- **Skills system** — 8 built-in skills, install API, agent composition
- **Agent registry** — versioned lifecycle with capabilities and permissions
- **Evaluation framework** — accuracy, cost, speed, reliability, tool usage benchmarks
- **Prompt workspace** — IDE for AI behavior
- **AI observability** — full execution traces
- **Cost intelligence** — BYOK token and spend tracking
- **buselligence.lock** — reproducible AI environments
- **Extension SDK** — `createBuselligencePlugin()` spec
- **CLI** — `bus create`, `bus add mcp`, `bus deploy`, `bus test agent`, `bus evaluate`
- **Hello World** — `/start` guided 60-second experience
- **Why Buselligence** — `/why` philosophical anchor page
- **Reference examples** — personal AI assistant, coding agent, BI analyst, autonomous business app
- **Documentation** — full docs product at `docs/README.md`
- **Contribution infrastructure** — issue templates, CONTRIBUTING, ROADMAP, DEVELOPMENT
- **Release automation** — GitHub Actions CI and release workflow
- **Powered by Buselligence** badge

### Changed

- Platform positioning: "open-source runtime for AI-powered applications"
- Health endpoint reports v8.0.0 with kernel feature flags
- `npm run setup` for one-command onboarding

## [7.0.0] - 2026-07
